# Steam-BI
Fluxo de engenharia de dados a partir de uma fonte de dados fria da Steam a fim de gerar inteligência de negócio (BI).

Utilizou-se da **arquitetura medalhão** para melhorar a qualidade dos dados incrementalmente e progressivamente.

Informação e conhecimento, derivados de **análises históricas e estatísticas** dos dados, foram por fim agregados em inteligência através de painéis de visualização no PowerBI.
